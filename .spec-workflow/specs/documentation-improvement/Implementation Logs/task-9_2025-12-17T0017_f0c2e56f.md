### Integrations

#### Integration

- **Description:** ax-layout-switcher integrates with application shell to switch between layout modes, storing preference in localStorage and emitting layoutChange events for dynamic layout rendering
- **Frontend Component:** ax-layout-switcher
- **Backend Endpoint:** None (client-side only)
- **Data Flow:** User clicks menu → Select layout → Update signal → Emit layoutChange event → Save to localStorage → Parent component applies layout CSS/routing

#### Integration

- **Description:** ax-gridster provides drag-and-drop grid functionality using angular-gridster2, supporting edit mode toggling, preset configurations, and real-time layout change events for dashboard persistence
- **Frontend Component:** ax-gridster
- **Backend Endpoint:** None (layout persistence handled by parent component)
- **Data Flow:** User drags/resizes item → Update item position → Emit itemChange event → Emit layoutChange event → Parent saves layout to backend/localStorage

#### Integration

- **Description:** ax-theme-switcher connects to AxThemeService for theme state management, automatically persisting selected theme to localStorage and applying CSS variables to document root
- **Frontend Component:** ax-theme-switcher
- **Backend Endpoint:** None (AxThemeService manages theme state)
- **Data Flow:** User selects theme → Call AxThemeService.setTheme() → Service updates signal → Service persists to localStorage → CSS variables applied to :root

#### Integration

- **Description:** ax-theme-builder provides full theme customization interface integrated with ThemeBuilderService, supporting color editing, M3 color generation, contrast validation, and multi-format export
- **Frontend Component:** ax-theme-builder
- **Backend Endpoint:** None (ThemeBuilderService manages theme configuration)
- **Data Flow:** User edits color/typography/spacing → ThemeBuilderService updates theme config signal → Real-time preview updates → User exports theme → ThemeBuilderService generates CSS/SCSS/JSON/Tailwind code → Download or copy to clipboard
