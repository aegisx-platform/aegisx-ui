### Integrations

#### Integration

- **Description:** Auth components integrate with backend authentication services via REST APIs with secure token handling and OAuth flows
- **Frontend Component:** ax-login-form, ax-register-form, ax-forgot-password-form, ax-reset-password-form
- **Backend Endpoint:** POST /api/auth/login, POST /api/auth/register, POST /api/auth/forgot-password, POST /api/auth/reset-password
- **Data Flow:** User submits form → Component validates → API call with credentials → Backend hashes password/validates token → Returns JWT tokens → Store in HttpOnly cookies (preferred) or localStorage (encrypted) → Navigate to authenticated route

#### Integration

- **Description:** QR code component integrates with angularx-qrcode library and supports download/clipboard APIs
- **Frontend Component:** ax-qrcode
- **Backend Endpoint:** N/A (client-side QR generation)
- **Data Flow:** User provides data/preset → Component generates QR code → Renders as canvas/SVG → User downloads or copies → Optional: Upload to backend for storage/tracking

#### Integration

- **Description:** Signature pad integrates with signature_pad library and Angular Forms ControlValueAccessor for seamless form integration
- **Frontend Component:** ax-signature-pad
- **Backend Endpoint:** POST /api/documents/{id}/signature
- **Data Flow:** User draws/uploads signature → Component emits SignatureData → Angular Forms captures value → Submit form → Backend validates and stores signature with audit trail → Returns signature ID
