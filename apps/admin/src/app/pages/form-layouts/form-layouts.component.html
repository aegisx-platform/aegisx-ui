Best Practices

  • check_circle Use .form-compact wrapper for consistent Tremor-style form sizing
  • check_circle Mark required fields with red asterisks (*)
  • check_circle Use mat-hint for helper text below form fields
  • check_circle Group related fields using grid layouts (grid grid-cols-1 md:grid-cols-2 gap-4)
  • check_circle Separate sections with mat-divider and adequate spacing (my-8)
  • check_circle Place primary action on the right, secondary actions on the left
