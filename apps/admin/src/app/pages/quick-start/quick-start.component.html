Best Practices

check_circle

Use Design Tokens

Always use CSS custom properties (design tokens) instead of hardcoded values for colors, spacing, and typography.

check_circle

Import Only What You Need

Take advantage of standalone components by importing only the specific modules you use in each component.

check_circle

Follow Material Design Guidelines

Adhere to Material Design 3 principles for spacing, typography, and component usage to maintain consistency.

check_circle

Use Semantic HTML

Always use proper HTML semantics and ARIA attributes for accessibility. Material components handle most of this automatically.
