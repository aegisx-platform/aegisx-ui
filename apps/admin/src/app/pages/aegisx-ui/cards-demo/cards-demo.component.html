‚ôø WCAG 2.1 Accessibility

Focus rings are automatically styled to match each card's color variant for better visual feedback. The focus outline uses a 2-layer design: inner layer (3px) matches the card's surface color, outer layer (5px) matches the card's primary color. Compliant with WCAG 2.1 Level AA focus indicators.
