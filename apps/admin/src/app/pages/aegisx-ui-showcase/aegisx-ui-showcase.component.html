üí° How it Works

The loading bar uses LoadingBarService for centralized control. Inject the service in any component to control the loading bar in the layout!

  • Service-based: Injectable via dependency injection
  • Reactive: Uses RxJS BehaviorSubject for state management
  • Type-safe: Full TypeScript support with interfaces
  • Flexible: Indeterminate & Determinate modes
  • Colors: primary, success, warning, error, neutral
