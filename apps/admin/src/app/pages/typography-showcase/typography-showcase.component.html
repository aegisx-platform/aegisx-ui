Prose Container Variants

Control the maximum width of your prose content 

<!-- Default: 65ch max-width -->
<div class="ax-prose">
  <p>Optimal reading width</p>
</div>

<!-- Wide: 75ch max-width -->
<div class="ax-prose ax-prose-wide">
  <p>Slightly wider for code docs</p>
</div>

<!-- Full: No max-width -->
<div class="ax-prose ax-prose-full">
  <p>Expands to container width</p>
</div>

Default (65ch)

The default .ax-prose container has a maximum width of 65 characters (~700-750px), the optimal line length for reading.

Wide (75ch)

The .ax-prose-wide variant increases to 75 characters, providing slightly more horizontal space while maintaining readability.

Full Width

The .ax-prose-full variant removes the width constraint entirely, allowing content to expand to fill its container.

Documentation Typography System

A clean, readable typography system inspired by shadcn/ui. Designed for technical documentation with high contrast and optimal readability.

Introduction

This typography system provides a complete set of components for building beautiful documentation. Every component is designed with readability in mind, using a black and white color scheme with carefully chosen spacing and sizing.

The system uses Inter for Latin characters and Noto Sans Thai for Thai text, with fallbacks to system fonts for optimal performance. Code examples use JetBrains Mono for excellent legibility.

รองรับภาษาไทย

ระบบตัวอักษรนี้รองรับภาษาไทยอย่างสมบูรณ์ด้วยฟอนต์ Noto Sans Thai ซึ่งออกแบบมาเพื่อความสวยงามและอ่านง่ายของตัวอักษรไทย

ตัวอย่างการใช้งาน

การออกแบบที่ดี เริ่มต้นจากการเข้าใจผู้ใช้ และ การเลือกเครื่องมือที่เหมาะสม เพื่อสร้างประสบการณ์ที่ดีที่สุด

หลักการออกแบบ

  • อ่านง่าย มีความคมชัดสูงระหว่างข้อความกับพื้นหลัง
  • ใช้ฟอนต์ที่เหมาะสมกับภาษาไทย (Noto Sans Thai)
  • ระยะห่างระหว่างบรรทัดที่เหมาะสม (line-height 1.7)
  • ขนาดตัวอักษรมาตรฐาน 15px สำหรับเนื้อหา
การออกแบบที่ดีไม่ใช่แค่การทำให้สวยงาม แต่เป็นการแก้ปัญหาและสร้างประสบการณ์ที่ดีให้กับผู้ใช้

  • H5: Small headings (15px, semibold)
  • H6: Smallest headings (14px, semibold)

Design Principles

Readability First

Every decision prioritizes readability. Font sizes, line heights, and spacing are optimized for comfortable reading over long periods.

High Contrast

The color scheme ensures maximum contrast and accessibility. Colors are used sparingly and intentionally.

Consistent Spacing

Vertical rhythm is maintained through consistent spacing using a base-4 scale (4px, 8px, 16px, 24px, 32px).

Optimal Line Length

Content is constrained to 65-75 characters per line for optimal reading experience.
