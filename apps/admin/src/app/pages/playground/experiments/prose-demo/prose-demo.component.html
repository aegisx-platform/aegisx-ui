Prose Container Variants

Different width options for prose content

Default (.ax-prose)

Maximum width: 90ch - ความกว้างที่เหมาะสำหรับ documentation และเนื้อหาทางเทคนิค

Standard Documentation Width

This container uses the default prose width of 90 characters, which is optimal for documentation with code examples and tables. ความกว้างนี้เหมาะสำหรับการอ่านเนื้อหา documentation รองรับตารางและตัวอย่าง code ได้ดี

Full Width (.ax-prose-full)

No maximum width - เต็มความกว้างของ container

Full Width Content

This container has no maximum width restriction and will expand to fill the available space. Best used for dashboards or layouts that need to utilize the full viewport width. เหมาะสำหรับหน้าจอใหญ่หรือเนื้อหาที่ต้องการพื้นที่เต็มที่
